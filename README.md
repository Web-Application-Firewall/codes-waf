## Description

This repository contains the supporting materials used in the experimental evaluation of open-source Web Application Firewalls (WAFs) under a comparative study designed for SME environments with non-dedicated hardware.

### openapi/
Contains the OpenAPI specification file used to define the DVWA endpoints evaluated by GoTestWAF. This file directs the tool to target specific application routes rather than performing generic evaluations.

### scripts/
Contains the Python script developed to process the CSV output generated by GoTestWAF. The script groups results by OWASP and community test suites, calculates the true negative rate (TN), and computes the weighted performance metric defined in the study.

## Usage

### Prerequisites

Some target applications require authentication to access their functionality.
In such cases, a valid session identifier must be obtained prior to running
GoTestWAF and passed as a request header.

For DVWA specifically, retrieve the session cookie by logging in through the
browser and inspecting the request headers, or by running:

```bash
curl -c cookies.txt -d "username=admin&password=password&Login=Login" \
  http:///login.php
```

Then extract the `PHPSESSID` value from `cookies.txt` and use it in the
command below.

### Running GoTestWAF

Replace `<TARGET_HOST>`, `<TARGET_PORT>` and `<PHPSESSID>` with the
appropriate values for your environment:

```bash
sudo docker run --rm --network="host" -it \
  -v ${PWD}/reports:/app/reports \
  -v ${PWD}/dvwa-openapi.yaml:/app/dvwa-openapi.yaml \
  wallarm/gotestwaf \
  --url=http://: \
  --openapiFile /app/dvwa-openapi.yaml \
  --reportFormat json,html \
  --addHeader "Cookie: PHPSESSID=;security=low"
```

> **Note:** The `security=low` cookie parameter is specific to DVWA and sets
> the application security level. Adjust or remove it if testing a different
> target application.

### Processing Results
Navigate to the `scripts/` directory and run:

```bash
cd scripts/
python3 process_results.py 
```

Or from the repository root:

```bash
python3 scripts/process_results.py
```
